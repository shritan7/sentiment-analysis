# AI Sentiment Analysis Web Application

## Overview

This project is an end-to-end Sentiment Analysis Web Application developed using Python, Flask, Scikit-learn, NLTK, HTML, CSS, and JavaScript.

The application predicts whether a user-entered review is Positive or Negative using a Machine Learning model trained on the IMDb Movie Reviews Dataset.

---

## Features

- Real-time sentiment prediction
- Text preprocessing using NLTK
- CountVectorizer for feature extraction
- Naive Bayes Machine Learning model
- Responsive Flask web interface
- Confidence score prediction
